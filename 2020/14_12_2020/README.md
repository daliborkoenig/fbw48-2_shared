## git

### lists

- To create a **local** repository:
  1. Git init
  2. add files (touch **readme.md** **.gitignore**)
  3. git status
  4. git add .
  5. git commit -m "message"
  6. we have to connect this repo with a remote repo
  7. git push to the remote repo
